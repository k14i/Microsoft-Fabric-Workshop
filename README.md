## 所用時間の計算方法

* サイト内に所要時間の表記がある場合、それを採用。
* そうでない場合、[Reading Time](https://chrome.google.com/webstore/detail/reading-time/nccohhimobidpghgpnejnbkpoichbbml) を使って算出。設定は 74 words per minute。
